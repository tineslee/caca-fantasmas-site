## Sobre o projeto

Este site foi desenvolvido como projeto profissional para os clientes **João Tocchetto** e **Rosa Maria Jaques**, fundadores da AMIPSO — uma organização dedicada à investigação de fenômenos paranormais no Brasil.

O objetivo foi criar um portal moderno e acessível para apresentar:

- A trajetória dos investigadores
- Conteúdo sobre paranormalidade, ufologia e espiritualidade
- Lançamentos editoriais e vídeos comemorativos
- Informações sobre consultas espirituais

---

> Todos os arquivos estão na raiz do repositório para compatibilidade com GitHub Pages.
